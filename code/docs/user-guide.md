# AI-PKM User Guide

## Introduction

Welcome to AI-PKM, an AI-first Personal Knowledge Management system that uses chat as the primary input method. This guide will help you understand how to use the application effectively to manage your knowledge.

## Getting Started

### Accessing the Application

1. Open your web browser and navigate to the application URL
2. Log in with your credentials or create a new account
3. You'll be directed to the home page with navigation options for Chat and Knowledge Graph

### Understanding the Interface

The application has three main sections:

1. **Home Page**: Overview of your knowledge base and recent activities
2. **Chat Interface**: Primary method for interacting with your knowledge
3. **Knowledge Graph**: Visual representation of your knowledge and its connections

## Using the Chat Interface

The chat interface is the primary way to interact with your knowledge base. You can use it to:

1. Create new knowledge nodes
2. Query your existing knowledge
3. Connect related pieces of information
4. Organize and structure your knowledge

### Creating Knowledge Through Chat

To create new knowledge nodes through chat:

1. Type your information in the chat input
2. The system will automatically extract structured data
3. You'll see a preview of the extracted knowledge
4. Confirm or modify the extracted structure
5. The information will be added to your knowledge base

Example:

```
You: I learned about a new machine learning technique called "Transformer architecture" that's used in large language models.

AI: I've extracted the following knowledge:
- Topic: Machine Learning
- Concept: Transformer architecture
- Related to: Large language models

Would you like me to save this to your knowledge base?

You: Yes, please.

AI: Knowledge saved! I've created a new node "Transformer architecture" and connected it to "Large language models".
```

### Querying Your Knowledge

To retrieve information from your knowledge base:

1. Ask questions in natural language
2. The system will search your knowledge base
3. Relevant information will be presented in the chat
4. Related nodes will be shown in the sidebar

Example:

```
You: What do I know about machine learning?

AI: Here's what I found in your knowledge base about machine learning:

1. Transformer architecture - A neural network architecture used in large language models
2. Gradient descent - An optimization algorithm used in training neural networks
3. Overfitting - A modeling error that occurs when a model is too complex

Would you like to explore any of these topics further?
```

### Agent Assistance

The system includes several AI agents that help manage your knowledge:

1. **Organization Agent**: Suggests ways to categorize and tag your information
2. **Connection Agent**: Identifies relationships between pieces of information
3. **Query Agent**: Helps answer questions about your knowledge
4. **Summary Agent**: Creates summaries of knowledge areas

Agent suggestions will appear in the chat interface. You can:

- Accept suggestions to apply them
- Reject suggestions you don't find useful
- Modify suggestions before applying them

## Knowledge Graph Visualization

The Knowledge Graph provides a visual representation of your knowledge:

1. Nodes represent individual pieces of information
2. Connections show relationships between nodes
3. Colors indicate different categories or tags

### Interacting with the Graph

- Click on nodes to view details
- Drag nodes to rearrange the visualization
- Zoom in/out to focus on specific areas
- Search for specific nodes using the search bar
- Filter the graph by tags, date, or other properties

### Creating Connections

To create connections between knowledge nodes:

1. Select a node in the graph
2. Click "Add Connection"
3. Select the target node
4. Choose the type of relationship
5. Add optional notes about the connection
6. Save the connection

## Tips for Effective Knowledge Management

1. **Be consistent** in how you structure information
2. **Use the chat interface** for quick capture of ideas
3. **Review agent suggestions** to improve organization
4. **Explore the knowledge graph** to discover connections
5. **Use tags** to categorize information
6. **Ask questions** to retrieve and connect knowledge

## Keyboard Shortcuts

- `Ctrl+N` (or `Cmd+N` on Mac): Create a new node
- `Ctrl+F` (or `Cmd+F` on Mac): Search
- `Ctrl+S` (or `Cmd+S` on Mac): Save current node
- `Esc`: Close current dialog

## Troubleshooting

If you encounter issues:

1. Refresh the page
2. Check your internet connection
3. Clear browser cache
4. Contact support if problems persist

## Privacy and Data Security

Your knowledge data is private and secure:

- All data is encrypted in transit and at rest
- Your knowledge base is only accessible to you
- You can export or delete your data at any time

For more information, please refer to our privacy policy.
